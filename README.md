## 🔄 Common tree Options

| Option | What It Does |
|--------|--------------|
| `tree` | Show full directory tree |
| `tree -L 2` | Limit depth level |
| `tree -a` | Include hidden files |
| `tree -h` | Show human-readable sizes |
| `tree -d` | Show directories only |
| `tree -f` | Show full file paths |

## 🚀 Quick Practice (Copy-Paste Ready)

```bash
# Show current directory structure
tree

# Limit depth to 2 levels
tree -L 2

# Show hidden files
tree -a

# Display file sizes
tree -h

# Show directories only
tree -d
```
